# 🏠 House Price Prediction using Linear Regression

## 📌 Project Description

This project was developed as part of the QSkill Python Development Internship.

It predicts house prices using Machine Learning based on features like area, bedrooms, bathrooms, stories, parking, furnishing status, and more.

---

## 🚀 Features

- Load housing dataset
- Data preprocessing
- Convert categorical data
- Train Linear Regression model
- Predict house prices
- Display model accuracy
- User input prediction

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
