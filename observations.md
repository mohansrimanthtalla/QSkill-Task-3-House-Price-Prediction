# House Price Prediction - Observations

## Objective
Developed a Linear Regression model to predict house prices based on different house features.

## Dataset
- Total Records: 545
- Total Features: 13
- Missing Values: 0

## Data Preprocessing
- Converted categorical values into numerical values.
- Split the dataset into training and testing data.

## Model
- Algorithm Used: Linear Regression
- Library: Scikit-Learn

## Result
- Model Accuracy (R² Score): 0.65

## Conclusion
The model successfully predicts house prices based on user input. The achieved accuracy demonstrates that Linear Regression can effectively estimate housing prices for this dataset.
